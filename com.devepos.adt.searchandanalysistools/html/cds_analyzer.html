CDS Analyzer

The CDS Analyzer offers the following types of analysis for a CDS view and/or Database Table/View
  • Top-Down Analysis (CDS View only)
  • Where-Used Analysis
  • Used Entities Analysis (CDS View only)
  • Field Analysis
To analyze a CDS view (or Database Table/View) perform one of the following actions:
  1. via the context menu of a CDS or Database Table editor
  2. via the context menu of a selected CDS View or Database Table/View in the Project Explorer
  3. via the context menu of a selected CDS View or Database Table/View in the Search View (if ABAP Object Search + was used)
