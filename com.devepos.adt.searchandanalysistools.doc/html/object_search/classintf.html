Search for Classes / Interfaces

The search type Class/Interface supports the following search filters:

Filter Description Negation Support
owner Author (Responsible Person)
package Development Package
desc Description
api Release State
type Restrict search to Class and/or Interface
cat Class Category (e.g. Exception)
flag Properties of a Class (e.g. Abstract)
lang Language Version (e.g. Unicode)
friend Global Friends of a Class
super Super Class
intf Using/Implementing Interfaces
meth Methods of Class/Interface
attr Attributes of Class/Interface

Search Examples

  • Search Filters: cat:exception,area_class / Maximum Results: 500
    Searches for all Shared Memory Area Classes and Exception Classes, also the maximum result count is raised to 500
  • Search Filters: attr:c_*=*constant*
    Searches for all Classes/Interfaces that have attributes that starts with c_ and whose initial value contains constant
  • Search Filters: meth:get_field,set_field / Use AND Search:
    Searches for all Classes/Interfaces that have the methods get_field and set_field
